# 🌍 Global Travel Trends & Insights Dashboard (2020–2024)

## 📌 Project Overview
This project analyzes **global traveller trends, spending, and preferences (2020–2024)** using interactive dashboards.  
It provides actionable insights for tourism stakeholders to optimize strategies, improve customer experience, and identify emerging travel patterns.

---

## 🎯 Objectives
- Identify **seasonal travel demand** patterns for promotions and resource allocation.  
- Assess **visa preferences** to streamline documentation support.  
- Track **year-over-year growth** for market expansion.  
- Measure **traveller satisfaction & recommendation likelihood**.  
- Analyze **booking channel usage** to refine distribution strategies.  
- Highlight **top destinations** to guide marketing campaigns.

---

## 🛠️ Tools & Techniques
- **Excel** → Data cleaning & preprocessing  
- **Power BI (Power Query + DAX)** → Data transformation, KPIs, and dashboard creation  

### 🔑 Key DAX Measures
- **Total Travelers:** 46K  
- **Total Travel Spend:** 115.50M  
- **Repeat Travelers:** 18K  
- **Solo Trips:** 2K  
- **Family Trips:** 11K  
- **Average Rating:** 3.94  

---

## 📊 Dashboard Features
- **Seasonal Demand Insights** → Bar Chart  
- **Visa Preference Breakdown** → Pie Chart  
- **Traveler Growth Trendline** → Line Chart  
- **Recommendation Sentiment Analysis** → Horizontal Bar Chart  
- **Booking Channel Performance** → Pie Chart  
- **Destination Popularity Index** → Bar Chart  

Interactive filters:  
- Origin Country  
- Traveller Type  
- Repeat Traveller  

---

## 🔍 Key Insights
- Seasonal demand remains **balanced across all seasons**.  
- **Visa-Free travel** dominates (30.47%).  
- **Online Travel Agencies** lead booking channels (35.36%).  
- **France & Egypt** emerge as top destinations.  
- Steady **growth trend (2020–2024)** reflects post-pandemic recovery.  

---

## 📈 Analytics Approach
- **Descriptive:** Balanced demand, visa-free dominance, steady growth.  
- **Diagnostic:** Driven by holidays, simplified entry, digital adoption.  
- **Predictive:** Growth beyond 2024, rise in mobile app bookings.  
- **Prescriptive:** Strengthen seasonal campaigns, expand visa partnerships, invest in digital booking apps.  

---
